# VigilantEye AI-Based Suspicious Activity Detection System
**An IoT-based AI system for real-time suspicious activity detection using YOLO, CNN, LSTM on Raspberry Pi with nightvision camera. Features multi-mode alerts, secure remote access, and selective storage**
</br>
</br>

## Features
- Real-time video monitoring
- AI-based suspicious activity detection
- Automated alerts (GUI, buzzer, email)
- Night vision support with IR LEDs
- User-friendly GUI for live feed monitoring
- Hardware integration (Raspberry Pi + sensors)
</br>
</br>

## Hardware Components
- Raspberry Pi 5 (8GB RAM)
- Infrared Camera
- IR LEDs for night vision
- LDR sensor
- Buzzer
- NRF24L01 module (optional)
